Our Mission

We strive to provide an inviting platform for emerging artists to establish their initial community on. By encouraging micro-donations on a per-artist basis, we hope to build a sort of crowdfunding platform that will allow artists to get the funding they need to succeed.

By working to keep operational costs to the bare minimum, we can ensure that the maixmum proceeds can go to our community.

If, ever, you have an issue with the way we are operating this service, please reach out. This service is provided for the comunity (Feathercoin and beyond) to support new artists as they explore their creative ability.

Road Map

  • Create and Upload songs using API
  • Serve songs from S3
  • User Creation and Authentication using Web interface
  • Song Creation and Deletion using Web interface
  • Song Updating
  • Play Count Implementation
  • Integrate Feathercoind Payment System
  • Interface overhaul
